Fill in the missing values

Now that we know the magnitude of missing values in our dataset we can construct a method to fill in these missing values. I chose to fill in the null values with the mean value for that interval #### Mean Steps by Interval

dfIntervalMean <- sqldf("
select interval
  ,avg(steps) as avg_steps
from df
group by 1")

head(dfIntervalMean)
##   interval avg_steps
## 1        0 1.7169811
## 2        5 0.3396226
## 3       10 0.1320755
## 4       15 0.1509434
## 5       20 0.0754717
## 6       25 2.0943396

Now that we have the mean steps by interval we can go about filling in the missing values. I have chosen to loop through each row in the dataframe but this can be vectorized for an improvement in speed.

dfTemp <- df

for(i in 1:nrow(dfTemp)) {
  for (j in 1:nrow(dfIntervalMean)) {
    if(is.na(dfTemp$steps[i]) & dfTemp$interval[i] == dfIntervalMean$interval[j]) {
      dfTemp$steps[i] <- dfIntervalMean$avg_steps[j]
    }
  }
}

dfFilledNa <- dfTemp
